# <a name="manage-jobs-in-advanced-ediscovery-preview"></a>管理工作，在 [進階電子文件 （預覽）

以下是案例的所追蹤的進階電子文件 （預覽） 中中的 [**工作**] 索引標籤的工作清單。

| 工作類型 | 描述 | 適用於此工作的工作負載 |
| :- | :- | :- |
| 重新編製索引的 custodian 資料 | 當 custodian 加入至案例時，所有已局部編製索引的項目重新編製索引與進階電子文件編製索引，如需詳細資訊，請參閱[進階 custodian 資料編製索引](indexing-custodian-data.md)。 | 進階 eDiscovery |
| 估計搜尋結果 | 估計搜尋結果是建立和針對工作負載，例如 Exchange、 SharePoint 和商務用 OneDrive 執行搜尋的結果。  預估包括統計資料，例如大小與搜尋所傳回的項目數目。  如需有關搜尋的詳細資訊，請參閱[收集資料的進階電子文件中的案例](collecting-data-for-ediscovery.md)。 | 進階 eDiscovery |
| 準備搜尋預覽 | 準備搜尋預覽工作會建立和針對工作負載，例如 Exchange、 SharePoint 和商務用 OneDrive 執行搜尋的結果。  預覽搜尋結果的可用以協助判斷搜尋的功效。  如需有關搜尋的詳細資訊，請參閱[收集資料的進階電子文件中的案例](collecting-data-for-ediscovery.md)。 | 進階 eDiscovery |
| 將資料加入至工作集 | 當搜尋結果會新增至工作集，將資料加入至工作組工作即會發生。  如需有關管理工作集的詳細資訊，請參閱 <<c0>管理使用進階電子文件中的設定。 | 進階 eDiscovery |
| 將資料加入至另一個工作集 | 將資料新增到另一個使用者選擇要將資料從一個工作集新增至另一個時，會建立工作的工作集。 | 進階 eDiscovery |
| 將非 Office 365 資料加入至工作集 | 工作包括工作集新增非 Office 365 資料處理，並將資料新增至已從這可能是相關的情況下，用戶端電腦上傳工作集如需有關新增非 Office 365 資料的詳細資訊，請參閱[載入非 Office 365 資料到工作集](load-non-office365-data.md)。 | 進階 eDiscovery |
| 準備錯誤解決方法 |  | 進階 eDiscovery |
| 新增資料修復的工作集 | 新增修復的工作集資料，請參閱時錯誤的檔案會修復和載入回至 Office 365 進階電子文件探索工作集所建立的工作。  如需有關錯誤修復的詳細資訊，請參閱 <<c0>錯誤修復時處理資料。 | 進階 eDiscovery |
| 比較負載設定 | 比較負載集合可讓使用者檢查載入至工作集，如需負載集合之間的差異，請參閱[管理載入設定](manage-load-sets.md)。 | 進階 eDiscovery |
| 標記的文件 | 當工作集合中的多個文件標記時，就會發生標記的文件工作。  如需詳細資訊，請參閱 <<c0>標記文件中的工作集。 | 進階 eDiscovery |
| 執行分析 | 執行分析工作參照處理工作包括接近重複的識別碼，電子郵件執行緒分析和佈景主題分析分析的詳細資訊，請參閱[進階電子文件中的工作集合中的分析資料](analyzing-data-in-working-set.md)。 | 進階 eDiscovery |
| 準備匯出資料 | 準備資料匯出工作的工作集，如需有關匯出的資料匯出結果，請參閱[匯出案例資料進階電子文件中](exporting-data-ediscover20.md)。 | 進階 eDiscovery |
