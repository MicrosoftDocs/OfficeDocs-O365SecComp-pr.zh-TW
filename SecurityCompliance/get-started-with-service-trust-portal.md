# <a name="get-started-with-the-microsoft-service-trust-portal"></a>開始使用 Microsoft 服務信任入口網站

Microsoft 服務信任入口網站會提供各種內容、 工具及其他有關 Microsoft 安全性、 隱私與規範的作法。
  
## <a name="accessing-the-service-trust-portal"></a>存取服務信任入口網站

在服務信任入口網站包含關於 Microsoft 的實作控制項和其中保護我們雲端服務和客戶資料的程序的詳細資訊。若要存取某些服務信任入口網站上的資源，您必須已驗證的使用者身分登入您的 Microsoft 雲端服務帳號 （Azure Active Directory 組織帳戶或 Microsoft 帳戶） 和檢閱並接受 Microsoft規範資料的非洩漏合約。
  
在 [**現有客戶可以存取服務信任入口網站**[https://aka.ms/STP](https://aka.ms/STP)具有下列線上訂閱 （試用版或付費） 其中一項： 
  
- Office 365

- Dynamics 365

- Azure
    
 > [!NOTE]
 > 組織相關聯的 azure Active Directory 帳戶具有存取權的文件和規範管理員類似功能完整的範圍。建立個人使用 Microsoft 帳戶有限 Service 信任的入口網站內容的存取權。 
  
**新的客戶及評估 Microsoft online services 的客戶**
  
若要建立新的帳戶或建立的試用帳戶，請使用其中一種註冊格式 （也用於試驗的帳戶） 來取得 STP 存取。
  
- 註冊新的[Office 365 企業版的試用帳戶](https://go.microsoft.com/fwlink/p/?LinkID=507653)或新的[Office 365 企業版的試用帳戶](https://go.microsoft.com/fwlink/p/?LinkID=698279)

- 註冊新的[Dynamics 365 試用帳戶](https://go.microsoft.com/fwlink/?LinkId=252780)

- 註冊新的[Azure 的試用帳戶](https://go.microsoft.com/fwlink/?LinkId=722737)。 
    
當您註冊免費的試用版、 或訂閱時，您必須啟用以支援您的存取權 STP Azure Active Directory。

## <a name="my-library"></a>「 我的文件庫

使用 「 我的文件庫功能將文件和資源服務信任入口網站新增至 「 我的文件庫] 頁面。這可讓您存取在單一位置相關的文件。 若要新增到我的文件庫文件，按一下右邊的文件的 **...** 功能表，然後選取**新增至文件庫**。您可以藉由按一下一個或多個文件旁的核取方塊，然後按一下 [**儲存至文件庫**] 頁面的頂端到我的文件庫新增多個文件。

此外，通知功能可讓您設定到我的文件庫的電子郵件傳送給您每當 Microsoft 更新已加入到我的文件庫的文件。若要設定的通知，移至您我的文件庫並按一下 [**通知設定**。您可以選擇通知的頻率，並指定傳送通知您組織中的電子郵件地址。電子郵件通知包括已更新的文件的連結及更新的簡短描述。

也請注意我們找出您我的文件庫中有已更新在過去 30 天內，不論您開啟通知中的任何文件。更新的簡短描述也會顯示在工具提示。 

## <a name="starter-packs"></a>入門套件

入門套件的文件中有關 Microsoft 雲端服務的特定產業 Microsoft curated 組。目前服務信任入口網站會提供下列三個入門套件的財務服務的組織。這些入門套件可協助評估及評估安全性、 規範、 和 Microsoft 雲端中的隱私權，並提供相關指導，以協助高度規範的財務服務產業中實作 Microsoft 雲端服務的組織。

- **評估入門套件**-使用進行的財務服務組織的 Microsoft cloud 早期評估。

- **評估入門套件**-之後評估，檢查清單中使用和其他指導此入門套件可協助您評估安全性、 規範、 和隱私權相關的風險的組織。

- **稽核入門套件**使用者指導使用稽核控制項和其他工具可協助指引您的 Microsoft Cloud services 協助的方式實作此入門套件減少您的組織洩露風險。

若要存取這些入門套件，請前往**Service 信任的入口網站 > Industries & 區域 > 產業解決方案 > 財務服務**。您可以開啟或下載文件從入門套件或將其儲存至您我的文件庫。
