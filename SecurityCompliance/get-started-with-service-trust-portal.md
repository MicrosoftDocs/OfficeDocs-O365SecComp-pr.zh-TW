# <a name="get-started-with-the-microsoft-service-trust-portal"></a>開始使用 Microsoft 服務信任入口網站

Microsoft 服務信任入口網站提供各種不同的內容、 工具和 Microsoft 安全性、 隱私權與法規遵循相關的其他資源。
  
## <a name="accessing-the-service-trust-portal"></a>存取服務信任入口網站

服務信任入口網站包含 Microsoft 的實作的控制項和其中保護我們的雲端服務與客戶資料的程序的詳細資料。 若要存取某些服務信任入口網站上的資源，您必須已驗證的使用者身分登入與您的 Microsoft 雲端服務帳戶 （Azure Active Directory 組織帳戶或 Microsoft 帳戶） 和檢閱並接受 Microsoft符合性材料的保密協議。
  
在 [**現有的客戶可以存取服務信任入口網站**[https://aka.ms/STP](https://aka.ms/STP)以其中一個下列線上訂閱 （試用版或付費）： 
  
- Office 365

- Dynamics 365

- Azure
    
 > [!NOTE]
 > 與組織相關聯的 azure Active Directory 帳戶具有存取權的文件和功能像合規性管理員中的完整範圍。 建立供個人使用的 Microsoft 帳戶與服務信任入口網站內容的存取有限。 
  
**新的客戶和評估 Microsoft 線上服務的客戶**
  
若要建立新的帳戶，或建立試用帳戶，使用下列其中一個下列註冊表單 （也用於試用帳戶） 來取得在 STP 存取。
  
- 註冊新的[Office 365 商務版試用版帳戶](https://go.microsoft.com/fwlink/p/?LinkID=507653)或新的[Office 365 企業版試用帳戶](https://go.microsoft.com/fwlink/p/?LinkID=698279)

- 註冊新的[Dynamics 365 試用帳戶](https://go.microsoft.com/fwlink/?LinkId=252780)

- 註冊新的[Azure 試用帳戶](https://go.microsoft.com/fwlink/?LinkId=722737)。 
    
當您註冊免費試用版或訂閱時，您必須啟用 Azure Active Directory 以支援您的存取權的 stp （機器翻譯）。

## <a name="my-library"></a>我的文件庫

若要在服務信任入口網站上的文件和資源新增至您 「 我的文件庫] 頁面上使用 「 我的文件庫功能。 這可讓您存取與您在單一位置相關的文件。  若要將文件新增至您我的文件庫中，按一下 [文件的右邊的 **...** ] 功能表，然後選取 [**儲存至文件庫**。 您可以新增多份文件我文件庫，按一下一個或多個文件旁的核取方塊，然後按一下 [頁面頂端的 [**儲存至文件庫**。

此外，「 通知 」 功能可讓您設定您我的文件庫，以便電子郵件訊息傳送給您每當 Microsoft 更新已新增至您我的文件庫的文件。 若要設定的通知，移至您我的文件庫並按一下 [**通知設定**。 您可以選擇通知的頻率，並傳送通知給您組織中指定的電子郵件地址。 電子郵件通知包括已更新文件的連結及更新的簡短描述。

也請注意，我們會識別您我的文件庫中已更新，在過去 30 天，無論您開啟通知內的任何文件。 更新的簡短描述也會顯示在工具提示。 

## <a name="starter-packs"></a>入門套件

入門套件的 Microsoft 策劃套產業特定的 Microsoft 雲端服務的相關文件。 目前，在服務信任入口網站提供金融服務組織的下列三個入門套件。 這些入門套件可協助評估並評估安全性、 規範和隱私權 Microsoft Cloud 中的，並提供相關指導，以協助在高管制金融服務業中實作 Microsoft 雲端服務的組織。

- **評估入門套件**-使用 Microsoft cloud for 金融服務組織的最早評估。

- **評估入門套件**-評估之後使用檢查清單和其他指導此入門套件中以協助評估與安全性、 規範和隱私權相關的風險貴組織。

- **稽核入門套件**使用者在使用稽核控制項和其他工具可協助指引您實作的 Microsoft 雲端服務的方式，可協助指引此入門套件降低風險貴組織的風險。

若要存取這些入門套件，請前往**服務信任入口網站 > 產業 & 區域 > 產業解決方案 > 金融服務**。 您可以開啟或下載文件中的入門套件或將它們儲存到您我的文件庫。
