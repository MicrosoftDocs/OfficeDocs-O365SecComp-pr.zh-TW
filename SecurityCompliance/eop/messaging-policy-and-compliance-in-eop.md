# <a name="messaging-policy-and-compliance-in-eop"></a>EOP 中的訊息原則和規範

Microsoft Exchange Online Protection 提供有助於您管理電子郵件資料的郵件原則及符合性功能。
  
## <a name="messaging-policy-and-compliance-documentation"></a>郵件原則及符合性文件

|**主題**|**描述**|
|:-----|:-----|
|[EOP 中的稽核報告](auditing-reports-in-eop.md) <br/> |稽核報告會提供如何執行系統管理員角色群組報告 (其會顯示在特定時間範圍內您組織中的管理角色群組發生的變更) 和系統管理員稽核記錄 (其會保留組織中發生之所有設定變更的記錄) 的相關資訊。  <br/> |
|[Transport Rules](http://technet.microsoft.com/library/743bd525-0ca2-426d-b76c-b4a052bc8886.aspx) <br/> |提供傳輸規則的相關資訊，傳輸規則可賦予您將本身公司特有原則套用到電子郵件的彈性。  <br/> |
