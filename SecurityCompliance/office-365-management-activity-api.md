# <a name="office-365-management-activity-api"></a>Office 365 管理活動 API

Microsoft 提供 reporting services, 您可以用來取得有關 Office 365 租使用者的匯總交易資訊。 [Office 365 管理活動 API](https://docs.microsoft.com/office/office-365-management-api/office-365-management-apis-overview)使用行業標準 RESTful 設計和 OAuth v2 進行驗證。 這可讓您輕鬆開始體驗如何檢索資料, 並將資料 ingesting 至視覺化檢視和應用程式。 API 提供 Office 365 中使用者、系統管理員、作業及安全性活動相關資訊的資料摘要。 資料可出於法規目的而保留, 或與內部部署基礎結構或其他來源的記錄資料採購相結合。 這可協助為整個企業的作業、安全性及合規性建立監視解決方案。

Office 365 管理活動 API 提供來自 Office 365 和 Azure Active Directory 活動記錄的各種使用者、系統管理員、系統及原則動作及事件的相關資訊。 API 提供一致的審核架構, 其所有服務都有10個以上的欄位。 API 可讓組織在事件之間進行簡單的連線, 並啟用資料的原因的新方法。

許多獨立軟體廠商 (Isv) 與 Microsoft 合作, 並已建立以 API 為基礎的解決方案。 某些解決方案主要著重于 Office 365 資料和其他雲端提供者和內部部署系統的來來源資料, 以建立作業、安全性及規範相關活動的統一視圖。 

如需詳細資訊, 請參閱[Office 365 管理活動 API 參考](https://docs.microsoft.com/office/office-365-management-api/office-365-management-activity-api-reference)。
