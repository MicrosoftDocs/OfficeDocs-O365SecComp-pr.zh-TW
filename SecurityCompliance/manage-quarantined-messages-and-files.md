# <a name="manage-quarantined-messages-and-files-as-an-administrator-in-office-365"></a>以 Office 365 的系統管理員身分管理隔離的郵件和檔案

作為系統管理員, 您可以查看、釋放和刪除隔離的郵件, 並在 Office 365 中報告假正隔離的郵件。 您也可以針對 SharePoint Online、商務用 OneDrive 和 Microsoft 團隊, 查看、下載和刪除由事先威脅防護 (ATP) 捕獲的隔離檔案。 您可以設定原則, 讓 Office 365 篩選郵件, 並將其傳送至隔離區, 有幾個原因: 因為它們被識別為垃圾郵件、大宗郵件、網路釣魚郵件、包含惡意程式碼, 或是因為它們符合郵件流程規則。
  
根據預設，Office 365 會直接將網路釣魚郵件和包含惡意程式碼的郵件傳送至隔離區。 其他篩選的郵件會傳送至使用者的 [垃圾郵件] 資料夾, 除非您設定原則將其傳送至隔離區。
  
您必須擁有 Office 365 的全域系統管理員 (GA) 許可權, 或安全性與合規性中心的安全性系統管理員, 才能使用傳送給其他使用者的隔離郵件, 以及使用隔離的檔案。
  
> [!IMPORTANT]
>根據預設, 垃圾郵件、大量和網路釣魚郵件會在30天內保持隔離。 因為符合郵件流程規則而被隔離的郵件, 會保留在7天內隔離。 惡意程式碼郵件會保留在隔離區15天。 您可以在安全性&amp;與合規性中心的反垃圾郵件設定中自訂垃圾郵件隔離時間。 當 Office 365 從隔離區刪除郵件時, 就無法再取得該郵件。 如果您想要, 您可以在反垃圾郵件篩選原則中變更隔離郵件的保留期間。 如需詳細資訊, 請參閱設定本文中[的隔離保留期間](manage-quarantined-messages-and-files.md#BKMK_ModQuarantineTime)。 
  
## <a name="view-your-organizations-quarantined-messages"></a>查看組織的隔離郵件

1. 在您的 Office 365 組織中使用具有全域系統管理員許可權的公司或學校帳戶, 登入 Office 365, 然後[移至安全性與合規性中心](go-to-the-securitycompliance-center.md)。
    
2. 在左邊的清單中, 展開 [**威脅管理**], 選擇 [**檢查**], 然後選擇 [**隔離**]。
    
    > [!TIP]
    > 若要直接移至**** 安全性&amp;與合規性中心中的隔離頁面, 請使用此 URL: >[https://protection.office.com/?hash=/quarantine](https://protection.office.com/?hash=/quarantine)
  
    根據預設, 安全性&amp;與合規性中心會顯示所有已隔離為垃圾郵件的電子郵件。 郵件會根據收到郵件的**日期**從最新到最舊的順序進行排序。 每封**** 郵件也會顯示**寄件者**、主旨及到期日 (**到期**日)。 您可以按一下對應的欄標頭, 依序排序欄位。再次按一下欄標題以顛倒排序次序。 
    
3. 您可以查看所有隔離郵件的清單, 也可以透過篩選來減少結果集。 您只能對100個專案執行大量作業, 因此如果您有超過此數目, 篩選也可以協助減少結果集。 您可以從頁面頂端的 [篩選] 中選擇一個選項, 以快速篩選郵件中的單一隔離原因。 選項包括:
    
  - 識別為垃圾郵件的郵件
    
  - 郵件被隔離, 因為它符合郵件流程規則 (也稱為傳輸規則) 所設定的原則
    
  - 識別為大宗郵件的郵件
    
  - 識別為網路釣魚郵件的郵件
    
  - 郵件被隔離, 因為包含惡意程式碼
    
此外, 作為系統管理員, 您可以選擇篩選組織中的所有郵件, 或只篩選傳送給您的郵件。 使用者只能查看和使用傳送給他們的郵件。
  
您也可以篩選結果, 以尋找特定郵件。 如需秘訣, 請參閱在本文中[篩選結果並尋找隔離的郵件和](manage-quarantined-messages-and-files.md#BKMK_AdvSearch)檔案。 
  
在您找到特定隔離郵件之後, 按一下該郵件以查看其詳細資料, 並採取動作, 例如將郵件發佈給某人的信箱。
  
## <a name="view-your-organizations-quarantined-files"></a>查看組織的隔離檔案

1. 在您的 Office 365 組織中使用具有全域系統管理員許可權的公司或學校帳戶, 登入 Office 365, 然後[移至安全性與合規性中心](go-to-the-securitycompliance-center.md)。
    
2. 在左邊, 展開 [**威脅管理**], 選擇 [**檢查**], 然後選擇 [**隔離**]。 <br/>
    > [!TIP]
    > 若要直接移至**** 安全性&amp;與合規性中心中的隔離頁面, 請使用此 URL: >[https://protection.office.com/?hash=/quarantine](https://protection.office.com/?hash=/quarantine)
  
3. 根據預設, 頁面會顯示隔離的電子郵件。 若要查看隔離的檔案, 請在頁面頂端設定篩選, 以顯示**** 因**惡意**代碼而隔離的檔案。 您必須具有 Office 365 的系統管理員許可權, 才能使用隔離的檔案。 
    
4. 檔案會根據檔案被隔離的日期, 從最新到最舊的順序進行排序。 上次修改檔案的**使用者**、檔案投遞的來源**服務**、檔案名、**位置**、檔案**大小**, 以及到期日 (**到期**日) (也會列出每個檔案)。 **** 您可以按一下標頭來排序欄位;再次按一下欄標題以顛倒排序次序。
    
您可以查看所有隔離檔案的清單, 也可以透過篩選來搜尋特定檔案。 就像訊息一樣, 您只能對100個專案執行大量作業。 目前, 安全性&amp;與合規性中心可讓您查看及管理隔離區中的檔案, 因為這些檔案已被識別為包含惡意程式碼。 如需秘訣, 請參閱在本文中[篩選結果並尋找隔離的郵件和](manage-quarantined-messages-and-files.md#BKMK_AdvSearch)檔案。 
  
## <a name="to-filter-results-and-find-quarantined-messages-and-files"></a>篩選結果並尋找隔離的郵件和檔案
<a name="BKMK_AdvSearch"> </a>

視您的設定而定, 可能會有大量隔離的郵件和檔案。 若要尋找特定的郵件或檔案或一組郵件或檔案, 您可以根據各種額外資訊篩選隔離的專案。
  
1. 在 [**隔離**] 頁面上, 確定篩選器的頂端列設定為視需要顯示郵件或檔案: 
    
      - 若要搜尋檔案, 請設定篩選器, **** 以顯示因**惡意**代碼隔離的檔案。<br/>
    對於隔離的檔案, 頁面會顯示所有隔離的檔案, 而不只是您自己的檔案, 而不論您要顯示的內容為何。
    
      - 若要搜尋隔離的郵件, 請設定篩選器以顯示**所有**或**僅顯示我**的**電子郵件**。 針對最後一個篩選, 選擇您要尋找的隔離郵件類型。 您可以針對符合郵件流程規則 (**傳輸規則**)、**大量**郵件、**網路釣魚**郵件或包含**惡意**代碼的郵件, 搜尋已識別為**垃圾**郵件的已隔離郵件。
    
2. 在 [**排序結果依據**] 下, 從下拉式清單中選擇您要用來搜尋的篩選器或篩選器。 選項會根據您搜尋的是檔案或郵件而有所不同。 搜尋欄位目前不支援萬用字元。<br/><br/>對於檔案和郵件, 您可以選擇要依郵件或檔案傳送至隔離的日期進行篩選。 您可以指定日期或日期範圍, 包括時間。 您也可以根據到期日, 從隔離區刪除檔案或郵件, 或使用篩選器組合來篩選搜尋結果。 若要依到期日期進行搜尋, 請選擇 [**高級篩選**]。 在 [**到期**] 下, 您可以選取即將從隔離區中刪除的郵件、在**** 未來的48小時 (**未來2天**)、下一周 (**未來7天**), 或您可以選擇自訂時間間隔。<br/><br/>對於郵件, 您可以使用下列其他選項:
    
      - **郵件識別碼**。 當您知道郵件識別碼時, 可使用此來識別特定郵件。<br/><br/>
    例如, 如果特定郵件是由組織中的使用者傳送或預定的, 但它永遠無法到達目的地, 您可以使用郵件追蹤來搜尋郵件 (請參閱[執行郵件追蹤及查看結果](https://go.microsoft.com/fwlink/?LinkId=799737))。 如果您發現郵件已傳送至隔離區 (可能因為它符合郵件流程規則或被識別為垃圾郵件), 您就可以在隔離區中指定郵件識別碼, 輕鬆找到此郵件。 請務必包含完整的郵件識別碼字串。 其中可能包含角括弧 (\<\>), 例如:<br/>
    `<79239079-d95a-483a-aacf-e954f592a0f6@XYZPR00BM0200.contoso.com>`
    
      - **寄件者電子郵件地址**。 選擇依單一寄件者電子郵件地址進行篩選。 
    
      - **收件者電子郵件地址**。 選擇依單一收件者的電子郵件地址進行篩選。 
    
      - **** 主旨。 輸入您要尋找之電子郵件地址的主旨。 由於不支援萬用字元搜尋, 您必須使用郵件的整個主旨, 才能讓搜尋傳回結果中的郵件。 搜尋不區分大小寫。 
    
## <a name="view-details-about-quarantined-messages-and-files"></a>查看有關隔離郵件和檔案的詳細資料

當您選取 [隔離] 清單中顯示的專案時, 您會在安全性&amp;與合規性中心右側的 [**詳細資料**] 窗格中看到其屬性的摘要。

- **收到**。 收到郵件的日期和時間。 
    
- **** 主旨。 郵件主旨行的文字。

- **到期**。 會自動從隔離區刪除郵件的日期和時間。 
    
- **發佈至**。 已發行郵件的所有電子郵件地址 (如果有的話)。 
    
- 尚未**發行至**。 尚未發行郵件的所有電子郵件地址 (如果有的話)。 
    
 **針對隔離的檔案顯示的詳細資料**
  
- **** 檔案名隔離區中的檔案名。 
    
- **網站路徑**定義 Office 365 中檔案位置的 URL。 
    
- 偵測**到的日期/時間**檔案被隔離的日期和時間。 
    
- **到期**將從隔離刪除郵件的日期。 
    
- **** 偵測用來偵測檔案中惡意程式碼的方法。 這可以是 ATP (高級威脅防護) 或 Microsoft 的反惡意程式碼引擎。 
    
- **發行**描述是否已發行檔案。 
    
- **惡意程式碼名稱**檔案中偵測到之惡意程式碼的系列和名稱。 
    
- **檔識別碼**檔的唯一識別碼。 
    
- 檔案**大小**檔案大小 (KB)。 
    
- **組織**您組織在 Office 365 中的唯一識別碼。 
    
- **修改者**上次修改檔案之使用者的公司或學校帳戶。 
    
- 檔案**大小**檔案大小 (KB)。 
    
- **SHA256 雜湊**檔案的雜湊。 您可以使用這種方式來查閱其他信譽儲存區, 或調查您的環境中的其他位置。 
    
## <a name="managing-messages-and-files-in-quarantine"></a>管理隔離區中的郵件和檔案
<a name="BKMK_ManageQuarantinedItem"> </a>

在您選取一封或一群郵件之後, 您有數個選項可以管理隔離區中的郵件。
  
- 無任何反應。 如果您選擇不執行任何動作, Office 365 會在到期時自動刪除郵件。 根據預設, 垃圾郵件、大量、惡意程式碼、網路釣魚以及被隔離的郵件, 會保留在30天內隔離的郵件流程規則。 當 Office 365 從隔離區刪除郵件時, 就無法再取得該郵件。 如果您願意, 您可以在反垃圾郵件原則中設定 [**保留垃圾郵件 (天)** ] 設定, 以變更隔離郵件的保留期間。 如需詳細資訊, 請參閱設定本文中[的隔離保留期間](manage-quarantined-messages-and-files.md#BKMK_ModQuarantineTime)。 
    
- **查看郵件頭**選擇此連結可查看郵件頭文字。 若要深入分析標頭, 請將郵件頭文字複製到剪貼簿, 然後選擇 [ **Microsoft Message Header analyzer** ] 以移至遠端連線分析程式 (按一下滑鼠右鍵, 然後選擇 [在新索引標籤**中開啟]** , 如果您不想離開 Office365以完成此工作)。將郵件頭貼到 [郵件頭分析器] 區段中的頁面上, 然後選擇 [**分析標頭**]。
    
- **預覽郵件**可讓您查看原始或 HTML 版本的郵件內文文字。 在 HTML 視圖中, 會停用連結。 
    
- **下載郵件**或**下載**檔案。 選擇此選項可將郵件或檔案的複本下載到您的本機裝置。 您必須先確認您瞭解從隔離區下載專案所產生的風險, 才會允許您這樣做。 郵件會以 .eml 格式儲存至您指定的資料夾。 隔離的檔案會以其原始格式儲存。
    
- **刪除**如果您想要, 您可以立即刪除隔離的專案 (或一組專案), 而不是等待 Office 365 所設定的到期日。 若要刪除郵件或檔案, 請在 [隔離] 清單中選取專案, 然後選擇 [**刪除**]。 若要一次刪除多個專案, 請選取 [隔離] 清單中專案左邊的核取方塊, 然後在出現的 [**大量動作**] 頁面上, 選擇 [**刪除選取的郵件**] 或 [**刪除選取**的檔案]。
    
- **發行**釋放隔離的專案 (或一組專案), 並將專案報告為虛假隔離 (假陽性) 給 Microsoft。 
    
    若要釋放和報告單一郵件或檔案, 請在 [隔離] 清單中選取專案, 然後選擇 [**發行**檔案] 或 [**發行訊息**]。 在下一個頁面上, 確定已選取 [針對分析將**郵件傳送至 microsoft** ] 或 [**報告檔案至 microsoft 進行分析**]。 
    
    若要一次釋放多個專案, 請選取 [隔離] 清單中專案左邊的核取方塊, 然後在出現的 [**大量動作**] 頁面上, 選擇 [**發行**檔案] 或 [ **release 郵件**]。 在下一個頁面上, 確定已選取 [針對分析將**郵件傳送至 microsoft** ] 或 [**報告檔案至 microsoft 進行分析**]。 
    
當您要釋放訊息時, 請注意下列事項:
  
- 當您一次執行多個郵件的大量發行時, 會將郵件發佈給所有原先識別的收件者。 如果您只想要將郵件只發佈給特定收件者, 您必須一次一次釋放一封郵件, 並個別識別收件者。
    
- 一封郵件無法一次發行給相同的收件者。
    
- 當您將郵件發佈給多個收件者時, 只有先前未收到該郵件的收件者才會出現在可能的收件者清單中。
    
- 當您選擇報告 false 陽性時, 如果您所發行的郵件或郵件被隔離為垃圾郵件、大量、網路釣魚或包含惡意程式碼, 則該郵件也會向 Microsoft 垃圾郵件分析小組報告。 小組會評估和分析郵件, 根據分析結果, 可能會調整全服務的垃圾郵件內容篩選規則, 以允許郵件通過。
    
## <a name="setting-the-quarantine-retention-period"></a>設定隔離保留期間
<a name="BKMK_ModQuarantineTime"> </a>

您可以設定郵件和檔案在過期之前保留在隔離區中的時間長度。 根據預設, 隔離的專案會保留30天。 您可以為每個建立的原則設定此設定。 您也可以修改預設原則的值, 如本文所述。 
  
### <a name="to-modify-the-quarantine-retention-period-for-the-default-spam-filter-policy-in-the-security-and-compliance-center"></a>在安全性與合規性中心修改預設垃圾郵件篩選原則的隔離保留期間

1. 在您的 Office 365 組織中使用具有全域系統管理員許可權的公司或學校帳戶, 登入 Office 365, 然後[移至安全性與合規性中心](go-to-the-securitycompliance-center.md)。
    
2. 在左邊, 展開 [**威脅管理**], 選擇 [**原則**], 然後選擇 [**反垃圾郵件**]。 <br/>
    > [!TIP]
    > 若要直接移至安全&amp;規範中心中的 [**反垃圾郵件**] 頁面, 請使用下列 URL: >[https://protection.office.com/?hash=/antispam](https://protection.office.com/?hash=/antispam)
  
3. 選擇 [**自訂**] 以顯示 [**自訂設定**] 索引標籤。 
    
4. 展開 [**預設垃圾郵件篩選原則 (ALWAYS ON)** ] 列。 
    
5. 選擇 [**編輯原則**]。 預設垃圾郵件篩選原則的設定會出現在新的頁面中。
    
6. 展開 [**垃圾郵件] 和 [大量動作**]。
    
7. 在 [**隔離**] 底下的 [**保留垃圾郵件天數**] 文字方塊中, 輸入您想要讓 Office 365 保留隔離郵件和檔案的時間長度。 預設值為30天。 這也是最大值。 
    
8. 選擇 [**儲存**]。
